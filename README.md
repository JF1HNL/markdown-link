# markdown-link
現在のサイトをタイトル付きでリンクを生成するchrome拡張

## 使い方
開いているページのタイトルとリンクをクリップボードにコピーする拡張機能です。
また、一部の文字を選択することでその部分のハイライト表示のURLをクリップボードにコピーするすることもできます。
